# IMAGE_PROMPT_ENGINE_V4

## Purpose

Generate cinematic, story-driven image prompts that create a cohesive
documentary rather than isolated beautiful images.

## Core Principles

-   Story over aesthetics
-   Emotion over symbolism
-   Consistent protagonist
-   Visual continuity
-   Cinematic realism
-   Photorealistic output

## V4 Features

-   Character continuity across scenes
-   Shot planning (Wide, Medium, Close-up, Macro, POV, Drone,
    Over-the-shoulder)
-   Emotion-first composition
-   Literal + symbolic balance
-   Visual storytelling progression
-   Scene composition rules
-   Color script across the video
-   Lighting progression
-   Camera angle variation
-   Environmental continuity
-   Time-of-day consistency
-   Cinematic depth and framing

## Prompt Structure

1.  Scene intent
2.  Subject
3.  Environment
4.  Emotion
5.  Camera shot
6.  Lens/composition
7.  Lighting
8.  Color palette
9.  Cinematic details
10. Negative prompt

## Character Rules

-   Maintain the same protagonist where applicable.
-   Keep clothing, age and appearance consistent.
-   Use recurring environments when appropriate.

## Shot Variety

Avoid repeating the same composition. Mix: - Establishing - Close-up -
Medium - Macro - POV - Over-the-shoulder - Drone - Tracking -
Low-angle - High-angle

## Visual Narrative

Every image should naturally connect to the previous and next scene.

## Quality Rules

-   No generic prompts.
-   Avoid repetitive symbols.
-   Avoid repeating mist, candles, lakes, thrones, etc.
-   Match narration precisely.
-   Prioritize human emotion.
-   Documentary-grade realism.

## Outputs

-   scene-XXX-original.txt
-   scene-XXX-optimized.txt
-   image_prompt_debug.json

## Validation

-   Prompt uniqueness
-   Character continuity
-   Shot diversity
-   Style consistency
-   Scene relevance
-   Visual continuity
-   Composition quality
