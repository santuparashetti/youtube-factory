# SCRIPT_BRANDING_V1.md

# Script Branding & Channel Identity Specification

## Objective

Every script generated by the Script Writer Agent must reinforce the
**Atma Theory** brand while feeling natural, thoughtful, and cinematic.

The branding should never sound repetitive, promotional, or forced.

------------------------------------------------------------------------

# Preserve Existing Pipeline

Do NOT modify:

-   LangGraph workflow
-   Research pipeline
-   Script generation flow
-   Scene planning
-   Image generation
-   TTS pipeline
-   Subtitle generation

Only enhance the final script generation stage.

------------------------------------------------------------------------

# Standard Script Structure

Every generated script should follow this order:

1.  Hook
2.  Welcome
3.  Topic Introduction
4.  Main Story
5.  Reflection
6.  Practical Takeaway
7.  Closing Reflection
8.  Atma Theory Sign-off

------------------------------------------------------------------------

# Welcome Message

Immediately after the opening hook, insert a short branded introduction.

Preferred default:

> Welcome to Atma Theory... where ancient wisdom meets modern life.

Alternate naturally between variations such as:

-   Welcome to Atma Theory... ancient answers to modern questions.
-   Welcome to Atma Theory... where timeless wisdom helps us navigate
    today's world.
-   Welcome to Atma Theory... together we explore ideas that help us
    live with greater clarity.

Never repeat the exact same introduction every episode.

------------------------------------------------------------------------

# Topic Transition

After the welcome, smoothly introduce today's subject.

Examples:

-   Today's question is...
-   Today we'll explore...
-   Have you ever wondered...
-   Let's understand why...

------------------------------------------------------------------------

# Ending Structure

Every script should conclude with:

1.  Reflection
2.  Practical takeaway
3.  Mention of Atma Theory
4.  Signature closing

------------------------------------------------------------------------

# Signature Closing Library

Rotate naturally between:

-   Think deeper... live clearer.
-   Until next time... keep questioning, keep growing.
-   The answers you seek may already be within you.
-   Stay curious... stay aware.
-   See you in the next journey through Atma Theory.

Avoid repeating the same ending every episode.

------------------------------------------------------------------------

# Brand Voice

Always sound:

-   calm
-   reflective
-   compassionate
-   wise
-   conversational
-   cinematic

Never sound:

-   preachy
-   sales-driven
-   overly dramatic
-   like a marketing script

------------------------------------------------------------------------

# Call To Action

If included, place it naturally near the end.

Preferred example:

"If this perspective helped you see life a little differently, consider
joining us for the next journey."

Avoid aggressive "Like, Share & Subscribe" language.

------------------------------------------------------------------------

# Acceptance Criteria

Every generated script should:

-   Open with a compelling hook.
-   Include an Atma Theory welcome.
-   Transition naturally into the topic.
-   End with reflection rather than promotion.
-   Include an Atma Theory sign-off.
-   Rotate introductions and endings to avoid repetition while
    maintaining a consistent brand identity.
